# Reflection Questions - NASA Space Explorer

Please answer the following reflection questions to complete your project (10 pts each). These help you think critically about what you've built!

## Question 1: API Data Integration (10 pts)
**What challenges did you face when fetching data from the NASA APOD API, and how did you overcome them?**

*Hints to consider:*
- Did you encounter any errors initially? How did you debug them?
- What did you learn about how the API returns data?
- How did you handle different date ranges and variations in the API response?
- What would you do differently if you had to integrate a different API in the future?

---

## Question 2: User Experience & Accessibility (10 pts)
**How did you ensure your app is accessible and user-friendly for all visitors?**

*Hints to consider:*
- What accessibility features did you implement (keyboard navigation, ARIA labels, screen readers)?
- How does your loading state help communicate the app's status to users?
- What happens if a user encounters an error? Is the message clear?
- How could you improve accessibility further?

---

## Question 3: Problem Solving & Debugging (10 pts)
**Describe a specific bug or issue you encountered and walk us through how you solved it.**

*Hints to consider:*
- Did the gallery not display correctly at first?
- Were there issues with modal functionality or focus management?
- How did you use browser tools (console, DevTools) to identify problems?
- What would you test to ensure your fix works across different browsers?

---

## Grading Rubric for Reflections
- **Full Credit (10 pts)**: Thoughtful, detailed answer showing genuine reflection and learning
- **Partial Credit (5-9 pts)**: Good effort but lacking detail or depth
- **Minimal Credit (1-4 pts)**: Brief answer with limited reflection
- **No Credit (0 pts)**: No response or irrelevant answer
